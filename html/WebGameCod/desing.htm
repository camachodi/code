¿HOW THE VIDEOGAMES DESING WORKS?


Learn to use and add element’s to a videogame project using gamemaker studio.


Is necessary to remember that the person in charge of the workshop will be helping
The participants will be handled all the element’s for the game inside a project. The project must contain (character png’s and animations, scripts for the movement, collision system, win state detector, destruction and creation].


First the participant’s will learn to create a object and to insert into it the given blueprint for movement.


Second they will create a cube that will work as a wall and floor of the level.


Third they will be guided into creating a single screen level


Fourth they will create an object that will act as a finish line for the player.


Fifth they will implement his character to ensure that everything works and will be helped if not.


Sixth the participants will create a enemy.


Seventh the participants will be instructed in how to create animation’s for the moving elements of the game.


Lastly the participant’s will be told to iterate and play with the element’s of the game, to create more enemy’s and changing the map.
